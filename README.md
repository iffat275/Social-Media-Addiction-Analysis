# Social Media Addiction Analysis 📱

# Project Overview
Analysis of TikTok and Instagram addiction patterns across 100 global countries using Python, SQL, and Power BI.

# Datasets
- Screen Time Behaviour Dataset
- TikTok & Instagram 100 Global Countries Dataset

# Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Power BI

# Analysis Performed
- Platform wise usage comparison
- Country wise addiction scores
- Sleep hours vs addiction impact
- Dopamine dependency analysis
- TikTok vs Instagram daily usage
- Age wise addiction patterns

# Key Insights
- Tanzania and Argentina show highest addiction scores
- Higher TikTok usage correlates with higher addiction scores
- Less sleep = higher addiction score
- Dopamine dependency directly impacts addiction levels
